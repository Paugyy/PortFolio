Changer le mot de passe

      

        Le mot de passe est stocké sous forme de hash SHA-256 — illisible même en regardant le code source.
        Pour le changer, entrez le nouveau mot de passe ci-dessous et copiez le hash généré dans admin.html à la ligne const STORED_HASH.
      

      

        
        
      

      

        
      

      

        

          
          

            
            
          

        

        

          1. Copiez ce hash

          2. Ouvrez admin.html dans votre éditeur

          3. Remplacez la valeur de const STORED_HASH = '...' par ce hash

          4. Faites git add admin.html && git commit -m "Update pwd" && git push

⚠️ Sécurité

      
    
        
  • Le mot de passe est hashé en SHA-256 — illisible dans le code source
    • 
        
  • Après 5 tentatives échouées → blocage 15 minutes
    • 
        
  • La session expire à la fermeture de l'onglet
    • 
        
  • Idéalement, mettez votre repo GitHub en privé (Settings → General → Danger Zone → Change visibility)
    • 
        
  • Pour changer le mot de passe, utilisez l'onglet Mot de passe
